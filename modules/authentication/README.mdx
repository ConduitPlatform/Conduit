# Authentication Module

This module offers user authentication.<br />
It supports logins with local credentials as well as popular identity providers.<br />
TwoFactor authentication is also provided.

## Features ‚úîÔ∏è

- Local authentication with email/password
- Optional email verification on local strategy
- Forgot password / Reset password
- Phone authentication
- Facebook login with client credentials
- Google login with client credentials
- Twitch login with redirect strategy
- Figma
- GitHub
- Microsoft
- Slack
- Service account/ API key
- Optional 2FA supported on all strategies
- Optional Account linking supported when email is the same between providers
