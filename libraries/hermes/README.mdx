# Hermes

This library is meant to facilitate the creation of REST, GraphQL and WebSockets APIs.<br />
It handles registering request types, registering middleware and generating API documentation.

It is utilized by [Admin](../../packages/admin) and [Router](../../modules/router).

## Features ✔️

- REST (with Swagger)
- GraphQL
- WebSockets (via Socket.io)
- Support for middleware
- Auto-generated API documentation
