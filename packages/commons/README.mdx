# Conduit Commons SDK

This library provides a framework to bind all Conduit core packages in a safe way and facilitates their intercommunication.

Check out [Core](../core), [Admin](../admin) and [gRPC-sdk](../../libraries/grpc-sdk)
to find out more about Conduit's internals.

## Features ✔️

- Defines an Interface for all modular functionalities of Conduit
- Provides constructors for model declaration in Conduit
