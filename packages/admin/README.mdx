# Admin

This package utilizes [Hermes](../../libraries/hermes) so as to expose administrative APIs and
provide a way for Conduit modules to register administrative routes.

Check out [Core](../core), [Commons](../commons) and [gRPC-sdk](../../libraries/grpc-sdk)
to find out more about Conduit's internals.

## Features ✔️

- REST (with Swagger)
- GraphQL
- WebSockets (via Socket.io)
